# AI 导航网站规格说明
启动：npx serve -l 8080

打包部署
# 安装依赖
npm install

# 构建混淆
npm run build

# 开发模式
npm run dev


## 1. 概念与愿景

一个现代化的AI工具导航平台，以深邃的暗色调为基底，搭配霓虹渐变色彩点缀，营造出科技感与神秘感交织的视觉体验。界面简洁但信息丰富，让用户能快速发现和探索各类AI工具。整体风格参考赛博朋克美学，但保持专业克制。

## 2. 设计语言

### 美学方向
赛博朋克风格（Cyberpunk-lite）：深色背景配合霓虹色彩渐变，圆角卡片设计，微妙的光晕效果，营造科技未来感。

### 色彩系统
- **主色**: #6366F1 (靛蓝紫)
- **辅助色**: #8B5CF6 (紫色)
- **强调色**: #22D3EE (青色)
- **渐变**: linear-gradient(135deg, #6366F1, #8B5CF6, #22D3EE)
- **背景深**: #0F0F1A
- **背景卡片**: #1A1A2E
- **边框**: #2D2D44
- **文字主色**: #FFFFFF
- **文字次要**: #A1A1AA

### 字体
- 标题: "Inter", system-ui, sans-serif (font-weight: 700)
- 正文: "Inter", system-ui, sans-serif (font-weight: 400)
- 中文回退: "PingFang SC", "Microsoft YaHei", sans-serif

### 空间系统
- 基础间距单位: 8px
- 卡片圆角: 16px
- 按钮圆角: 8px
- 容器最大宽度: 1400px
- 卡片内边距: 24px

### 动效哲学
- 悬停卡片: translateY(-4px), box-shadow增强, 200ms ease-out
- 按钮悬停: 渐变位移, scale(1.02), 150ms
- 页面加载: 卡片依次淡入上移, 延迟100ms, 持续300ms
- 标签切换: 滑动指示器, 200ms ease-out

### 视觉资源
- 图标: Lucide Icons (via CDN)
- 工具图标: 使用彩色圆形背景+白色首字母
- 装饰: CSS渐变背景球体, 模糊光晕效果

## 3. 布局与结构

### 页面架构
```

│  Header: Logo + 导航菜单 + 搜索框                    │
├─────────────────────────────────────────────────────┤
│  Hero Section: 大标题 + 副标题 + 快捷搜索入口        │
├─────────────────────────────────────────────────────┤

├─────────────────────────────────────────────────────┤
```

### 响应式策略
- 桌面 (>1200px): 4列工具网格
- 平板 (768-1200px): 3列网格
- 手机 (<768px): 2列网格, 汉堡菜单

### 视觉节奏
- Hero区域: 大留白, 聚焦视觉效果
- 分类区域: 紧凑标签栏
- 工具区域: 卡片网格, 信息密度适中
- Footer: 简洁收尾

## 4. 功能与交互

### 核心功能
1. **分类浏览**: 点击分类标签筛选显示对应工具
2. **工具搜索**: 实时搜索过滤工具名称和描述
3. **快捷入口**: 顶部提供常用AI平台快捷链接
4. **工具卡片**: 显示图标、名称、简介、标签

### 交互细节
- **分类标签点击**:
  - 点击切换active状态
  - 标签下方滑动指示器跟随
  - 工具列表平滑过渡筛选
- **工具卡片悬停**:
  - 卡片上浮4px
  - 边框渐变发光效果
  - 光标变为pointer
- **搜索框**:
  - 聚焦时边框发光
  - 输入时实时过滤(300ms防抖)
  - 空结果显示友好提示
- **快捷链接悬停**: 渐变色位移效果

### 边界情况
- 空搜索结果: 显示"未找到匹配的工具"提示
- 图片加载失败: 显示彩色首字母背景
- 长文本: 单行截断+省略号
